# TangTech 网站构建笔记

## 竞品分析：SetupClaw (setupclaw.com)
- 定位：白手套服务，面向创始人/高管团队
- 价格：一次性安装费 + 月度托管
- 特点：Composio OAuth、Docker 沙盒、14天护航期
- 目标客户：4-50人团队、CEO/CFO/销售总监
- 媒体背书：MacStories、Fast Company、IBM、Cisco

## TangTech 差异化定位
- **主打自然语言安装**：用户说人话描述需求，我们自动生成配置
- **面向中文用户**：中国创业者、独立开发者、自由职业者
- **更低门槛**：个人用户也能用，不只是企业团队
- **自动化程度更高**：AI 辅助配置，减少人工干预

## 网站结构
- index.html：主页（中文默认，英文切换）
- scripts/：构建脚本和工具
